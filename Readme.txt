This is a simple todolist webapp using react for front end and local storage for backend instead of connecting it to any database thus simplifying the code 
Few commands that need to be run :
$npm install //to install the node modules
$npm i u uid 
//the commands below are for istalling differnt font styles 
$npm i --save @fortawesome/fontawesome-svg-core
$npm i --save @fortawesome/free-solid-svg-icons
$npm i --save @fortawesome/free-regular-svg-icons
$npm i --save @fortawesome/react-fontawesome@latest
